# Design System

## Estilo general
Interfaz limpia, simple y orientada a dashboard.  
Diseño moderno con componentes claros, buen contraste y enfoque en legibilidad.  
Se prioriza experiencia funcional sobre elementos decorativos.

## Tipografía
- system-ui, Arial, sans-serif
- Tamaño base: 16px
- Encabezados en peso semibold o bold

## Paleta de Colores

- Fondo principal: #F4F5F7
- Superficie (cards): #FFFFFF
- Texto principal: #2F3A4A
- Primario (acciones): #405B85
- Borde: #E5E7EB
- Éxito: #A7D7C5
- Advertencia: #F3D9A3
- Error: #FCA5A5 

## Componentes
- Navbar simple con nombre de la app.
- Cards KPI.
- Tabla de productos / sabores.
- Formulario con labels visibles.
- Botones:
  - Primary (acciones principales).
  - Secondary (acciones secundarias).
- Badge de estado para pedidos:
  - Confirmed
  - NeedsAction
  - Delivered (futuro)

## Reglas
- Bordes redondeados (12px)
- Espaciado consistente (8/12/16/24)
- Foco visible en inputs y botones (accesibilidad)
- Contraste suficiente entre fondo y texto
- Mobile First

## Principios UI

- Simplicidad antes que animación.
- Información clara y directa.
- Feedback inmediato al usuario.
- Estados visuales diferenciados (éxito, error, advertencia).
