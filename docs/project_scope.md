# Project Scope — Inventario y Ventas (MVP)

## Objetivo
Crear una app web simple para gestionar productos y stock, y registrar ventas básicas.

## Usuarios
- Vendedor: consulta productos y registra ventas.
- Administrador (futuro): gestiona usuarios, reportes y configuración.

## Funcionalidades (MVP — Fase 1: UI)
- Ver resumen (tarjetas KPI): productos, stock total, ventas del día.
- Ver lista de productos en una tabla (datos de ejemplo).
- Formulario para agregar producto (solo interfaz, sin lógica aún).

## Fase 2 (JavaScript)
- Agregar/editar/eliminar productos (en memoria y/o LocalStorage).
- Búsqueda y filtros por nombre/categoría.
- Validaciones y mensajes amigables (errores/éxito).

## Fase 3 (React + Firebase)
- Autenticación (roles).
- Base de datos en Firestore.
- Historial de ventas y reportes.
- Despliegue (Vercel/Firebase Hosting).

## Pantallas (inicial)
- Home / Dashboard (una sola página en Fase 1).
